📊 Estimating Descriptive Statistics of Financial Statements using Python

As part of my data analytics coursework, I analyzed the financial statement data of Ashok Leyland using Python in Google Colab.

The dataset was imported as a CSV file and key financial variables such as Equity Capital, Reserves, and Total Assets were analyzed using descriptive statistics.

🔎 Steps performed:

Imported the CSV dataset using pandas

Selected financial variables from the balance sheet

Estimated descriptive statistics using df.describe()

📈 Statistical measures obtained:

Count

Mean

Standard deviation

Minimum and maximum values

Quartiles (25%, 50%, 75%)

💡 Insight:
Descriptive statistics provide a quick summary of the financial health and distribution of financial variables across different years.
